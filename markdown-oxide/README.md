# Markdown Oxide

PKM (Personal Knowledge Management) Markdown language server for Claude Code with wikilink and Obsidian-style support.

## Supported File Extensions

- `.md` - Markdown files

## Installation

Using Homebrew:
```bash
brew install markdown-oxide
```

Using Cargo:
```bash
cargo install markdown-oxide
```

Using winget (Windows):
```bash
winget install --id=FelixZeller.markdown-oxide -e
```

Download pre-built binaries from [GitHub Releases](https://github.com/Feel-ix-343/markdown-oxide/releases).

## Server Command

```bash
markdown-oxide
```

## Features

- Wikilink completions (`[[link]]`)
- File and heading completions
- Tag support
- Go to definition
- Find references
- Document symbols
- Hover preview
- Rename support
- Obsidian vault compatibility
